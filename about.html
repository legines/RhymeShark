Ghostwriting rhymes since 2018

Have you ever thought "Hey I want to be the best sound cloud rapper the world has ever seen?" Well, look no further. Rhyme Shark is a state of the art technology used to help generate the perfect lyrics to make your next sound cloud track be blasted by every high schooler in a 2003 Honda civic across the country. Rhyme Shark is also success proven it has been used to create masterpiece song such as Lil Pump's top hits, Drose and Boss.

This app aims to help users create lyrics by using the Words API to match a searched word with rhymes. The Musixmatch API is used to present artists, song titles, and lyrics that contain the searched word. Users will have the opportunity to apply filers (mostly popularity asc/desc) to their searches. This is used to show the user examples of how their word of interest is used by other artists.
